# Request
#Actions/1 
#Ability-tag/Auditory  #Ability-tag/Concentrate  #Ability-tag/Linguistic  #Ability-tag/Mental 

Roll a **Diplomacy Check** to make a request of a creature that's [[Friendly]] or [[Helpful]] to you.

* **Critical success:** The target agrees to the request straight away.
* **Success:** The target agrees to the request but may ask for something in return or alterations to the request.
* **Failure:** The target refuses the request but may propose a less extreme alternative
* **Critical failure:** The target refuses the request and their attitude toward you decreases by one step.
